# 3. The Bohmian wave must live in configuration space, not as ordinary pressure

The attached file says the pilot wave is like a stress or pressure wave through the E8 condensate. That is vivid, but physically risky.

In Bohmian mechanics, the wavefunction is not usually a simple wave in ordinary 3D space. For many particles, it lives on **configuration space**. For fields, it is a wave-functional over field configurations.

So if the pilot wave is treated as a literal pressure wave moving through a medium, you risk reducing a nonlocal quantum object into a local classical wave. That breaks the very thing Bohmian mechanics was meant to preserve.

The stronger version is:

> The condensate is not ordinary physical medium; it is a relational state-space medium. The pilot wave is a wave-functional over admissible configurations of the condensate. The apparent local wave in spacetime is a projected, coarse-grained shadow of that deeper wave-functional.

So:

```text
Weak version:
pilot wave = pressure wave in E8 medium

Stronger version:
pilot wave = wave-functional on E8-constrained relational configuration space

Observed wave behavior =
projection of that wave-functional into emergent spacetime
```

This keeps Bohmian nonlocality intact while still giving the wave a concrete ontology.

# 4. Lorentz invariance is not solved by statistical averaging alone

The attached file proposes that Lorentz invariance emerges because quantum equilibrium statistically masks microscopic anisotropy. That is suggestive, but insufficient.

Statistical isotropy is not enough. Lorentz invariance is much stronger than “directions average out.” You need:

```text
same limiting speed for all low-energy species
relativistic dispersion relations
suppression of Lorentz-violating operators
no detectable preferred frame
stable light cones
covariant effective field equations
```

The sound-wave analogy is useful but dangerous. Sound waves in a fluid can obey a Lorentz-like equation, but the fluid still has a rest frame. A real TOE cannot merely say “all observers fail to notice the ether.” It must explain why every internal rod, clock, particle, and interaction couples to the same emergent metric.

The stronger resolution:

> Lorentz invariance must emerge as an attractive RG fixed point, not merely as statistical smoothing.

So the revised claim should be:

```text
Microscopic level:
relational substrate may not be Lorentz invariant.

RG flow:
Lorentz-violating operators become irrelevant.

Infrared fixed point:
all low-energy excitations share one effective causal cone.

Failure condition:
any surviving relevant Lorentz-violating operator kills the bridge.
```

That is much stronger.

Until that branching table exists, this is not yet a solved gap. It is a promising target.

# 7. Holography is promising, but AdS/CFT cannot be imported wholesale

The attached file uses holography and AdS/CFT as part of the emergence story. That is good, but it must be guarded.

AdS/CFT is best understood in special settings: asymptotically anti-de Sitter spacetimes, conformal boundary theories, supersymmetric examples, large-N limits, and related structures. Our universe is not simply AdS.

So the stronger version is:

```text
Do not claim:
AdS/CFT directly describes our universe.

Claim instead:
holography supplies a bridge template:
boundary degrees of freedom ↔ emergent bulk geometry,
RG scale ↔ radial/emergent dimension,
entanglement ↔ geometry.
```

That lets holography act as a plugin rather than a forced foundation.
